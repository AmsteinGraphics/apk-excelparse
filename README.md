# apk-excelparse

A lightweight Android app that can read and write values in Excel files.

## What changed
- Replaced the Flutter scaffold with a simple Android-native project.
- Added a GitHub Actions workflow that builds an Android debug APK on every push.
- Added a Settings screen action that checks the latest GitHub release and opens the release page for installation.

## Local development
1. Install Android Studio and the Android SDK.
2. Open the project in Android Studio.
3. Build or run the app from the Android Studio UI.
4. Open the main screen to use the update-check action.
