# apk-excelparse

A lightweight Flutter app that can read and write values in Excel files.

## What changed
- Added a GitHub Actions workflow that runs Flutter dependency installation, analysis, and APK compilation on every push.
- Added a Settings screen entry that lets the app check the latest GitHub release and open the update page for installation.

## Local development
1. Install Flutter.
2. Run `flutter pub get`.
3. Launch the app with `flutter run`.
4. Open Settings to use the update-check action.
